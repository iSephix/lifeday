# Contributing to Lifeday

- Use issues to propose schema changes.
- For breaking changes, propose a PR that bumps `meta.schema_version`.
- Extension naming: use `vendor:name` or `domain:name`, e.g. `oura:raw`, `labs:cbc`.
- Add examples and tests for any new fields.
